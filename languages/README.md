# Translation Files

This directory contains translation files for the Prospergenics theme.

## How to Translate

1. Install a translation plugin like Loco Translate or Poedit
2. Create a new translation for your language
3. Translate all strings
4. Save the .po and .mo files in this directory

## File Naming Convention

Translation files should follow this pattern:
- `prospergenics-{locale}.po` (editable translation file)
- `prospergenics-{locale}.mo` (compiled translation file)

Examples:
- `prospergenics-nl_NL.po` and `prospergenics-nl_NL.mo` (Dutch)
- `prospergenics-sw.po` and `prospergenics-sw.mo` (Swahili)
- `prospergenics-fr_FR.po` and `prospergenics-fr_FR.mo` (French)

## Text Domain

All translatable strings use the text domain: `prospergenics`

## Tools

Recommended translation tools:
- **Loco Translate** (WordPress plugin) - easiest for beginners
- **Poedit** (desktop app) - more features for professional translators
- **WP-CLI** - command line tool for developers
