# Theme Images

This directory contains background images for the parallax sections.

## Required Images

For best results, add the following images:

1. **hero-bg.jpg** (1920x1080px)
   - Main hero section background
   - Should feature team members or community gathering
   - Can be set via Customizer

2. **about-bg.jpg** (1920x1080px)
   - About section parallax background
   - Suggested: Diverse group collaborating

3. **programs-bg.jpg** (1920x1080px)
   - Programs section parallax background
   - Suggested: Learning/training environment

4. **contact-bg.jpg** (1920x1080px)
   - Contact section parallax background
   - Suggested: Welcoming community scene

## Image Specifications

- **Format**: JPG or PNG
- **Minimum Resolution**: 1920x1080px
- **Recommended Resolution**: 2560x1440px for Retina displays
- **File Size**: Optimize to <500KB per image for performance
- **Aspect Ratio**: 16:9

## Placeholder Images

If specific images are not available, you can use:
- Solid color backgrounds (set in CSS)
- Gradient backgrounds (already defined as fallbacks)
- Stock photos from Unsplash, Pexels, or Pixabay (free, high-quality)

## Screenshot

Add **screenshot.png** (1200x900px) to the theme root directory for the theme preview in WordPress admin.
